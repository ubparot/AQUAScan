Manual control features include:  ●   Virtual joystick steering and throttle control  ●   Dead-zone filtering for more stable handling  ●   Differential drive mixing for two-motor steering  ●   Real-time Wi-Fi communication between interface and vessel  ●   Live motor-output feedback during operation  Semi-Autonomous Navigation  AquaScan also supports semi-autonomous operation through its GPS module and Unity-based mission interface. Instead of driving entirely by hand, the operator can select specific GPS sampling points within the Unity interface. Once those locations are chosen, the software compares the boat’s current position to the target coordinates and adjusts motor output so the vessel moves toward the correct point.  This makes the system especially useful for repeated environmental monitoring because it allows sampling to occur at more controlled and consistent locations. That matters in scientific fieldwork because repeated measurements are much more useful when they are taken at known positions rather than at random or approximate locations.  The operator can still use manual control when needed, so the system is not locked into one mode. This gives AquaScan flexibility in real field conditions, where obstacles, wind, or changing water conditions may require direct intervention.  Semi-autonomous navigation allows AquaScan to:  ●   Travel to selected GPS waypoints  ●   Pause at specific monitoring locations  ●   Support repeatable sampling at known coordinates  ●   Combine autonomous guidance with manual override when needed  Sampling Process  Once AquaScan reaches a sampling location, the vessel is intended to stop and hold position while the probe is lowered to selected depths. This is important because collecting data while stationary improves measurement consistency and helps the probe move vertically through the water column instead of trailing behind the vessel.  The winch system is controlled through the same general software structure used for motor control. Instead of sending left and right drive outputs, the program commands the winch motor to raise or lower the tethered probe. Probe depth is estimated from spool rotation, using the spool geometry to convert rotation into cable length. This allows the system to position the probe at selected depths without relying on a pressure sensor.  If multiple depth readings are required, the software can lower the probe to one depth, allow it to stabilize, collect the sensor data, and then move it again. This lets AquaScan build a vertical profile of the water column rather than collecting only surface measurements.
