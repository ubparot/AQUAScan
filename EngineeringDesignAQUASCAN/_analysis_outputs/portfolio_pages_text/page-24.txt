What this stage showed:   Pond testing demonstrated that AquaScan could function beyond an artificial pool environment and perform the main task it was designed for. The vessel operated in natural calm-water conditions, collected data from 20 sampling points, used the probe to measure depth-based conditions, and connected the results to mission data through the software system. This test provided the strongest evidence that AquaScan is practical for ponds, lakes, marinas, sheltered creeks, and protected nearshore environments. It also showed that the system could be expanded into more advanced field testing, including longer sampling routes, repeated monitoring, and future environmental partnerships.  Stage 4: Future Testing and Validation  Although pool and pond testing verified AquaScan’s main functions, additional testing would be needed before using the system for larger-scale environmental monitoring. Future testing would focus less on proving that the prototype works and more on proving that it can collect useful environmental data consistently over time.  Purpose of future testing:   Evaluate AquaScan in larger and more environmentally meaningful locations while increasing route length, sampling frequency, autonomous operation, and real-world data value.  Future testing plan:  ●   Larger calm-water environments:   The next testing locations would include a marina, sheltered creek, larger lake, or protected section of the Chesapeake Bay. These sites would provide more realistic environmental variation while still matching AquaScan’s intended operating range.  ●   Repeated sampling routes:   AquaScan would be tested on preset routes where it collects readings from the same locations over multiple runs. This would help show whether the system can produce consistent data and track changes over time.  ●   Expanded autonomous operation:   Future tests would use GPS waypoint navigation more heavily instead of relying mainly on manual control. This would better evaluate AquaScan’s ability to complete monitoring routes with less operator input.  ●   Stronger environmental comparison:   Future testing would compare readings across different locations, depths, times of day, and weather conditions. This would make the data more useful for identifying trends, localized problem areas, and changes in water quality.  ●   Partnership-based validation:   A longer-term testing goal would be to work with a marina, restoration group, environmental organization, or local agency. This would allow AquaScan to collect data from a site where the results could support real monitoring, cleanup planning, or restoration decisions.
